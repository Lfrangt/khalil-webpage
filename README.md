# Khalil 个人主页

这是我的个人主页项目，使用 HTML、CSS 和 JavaScript 构建。

## 特点

- 响应式设计
- 平滑滚动效果
- 渐入动画
- 社交媒体链接
- 技能展示
- 联系方式

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- Font Awesome 图标

## 本地运行

1. 克隆仓库
```bash
git clone https://github.com/你的用户名/khalil-website.git
```

2. 打开 `index.html` 文件即可在浏览器中查看
